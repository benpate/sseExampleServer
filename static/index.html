SSE Placeholder

SSE Placeholder is a simple service that generates Server Sent Events for your test pages to read. It streams fake data from jsonplaceholder to your website on a semi-regular schedule every 0.5 - 2.0 seconds.

Server Sent Events in JSON

  • /posts.json
  • /comments.json
  • /albums.json
  • /photos.json
  • /todos.json
  • /users.json

Server Sent Events in HTML

  • /posts.html
  • /comments.html
  • /albums.html
  • /photos.html
  • /todos.html
  • /users.html

How Does It Work?

When you open one of the URLs below, this little server generates new Server Sent Events that are delivered to your test pages in either JSON or HTML format at random intervals. You can also specify a few options when you connect, using URL parameters:

No Event Name Do not specify an event name to test "onmessage" event handlers
/posts.html
One Event Name Specify a single event name to test one named event handlers
/posts.html?eventType=Name1
Multiple Event Name Specify a multiple event names to test several named event handlers. The server will randomly choose an name for each new record.
/posts.html?eventType=Name1,Name2
